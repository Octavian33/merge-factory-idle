# Merge Factory Idle

Unity 2022+ prototype project scaffolded from scratch.

## Open
1. Open Unity Hub
2. Add project: `Merge Factory Idle`
3. Use Unity 2022.3 LTS or newer
4. Press Play in any scene (bootstrap auto-builds full game scene)

## Features Included
- Buy workers with coins
- Drag & drop merge identical workers
- Scaled worker income
- Infinite loop with prestige reset
- Save/load to `Application.persistentDataPath`
- Offline earnings (up to 8h cap)
- Upgrade economy buttons
- Floating coin text and merge particles
- Portrait-friendly UI layout
- Android-friendly lightweight setup

## Notes
- Uses only built-in Unity systems and runtime-generated placeholder art.
- No existing project files are modified.
